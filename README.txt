backendproj

Посилання на додаток: https://backendprojlab.herokuapp.com

Щоб отримати список категорій потрібно виконати запит: https://backendprojlab.herokuapp.com/categories

Щоб отримати список записів по певному користувачу, потрібно виконати POST request, щоб отримати по якому користувачеві
хочемо дивитись а тоді виконати запит:
https://backendprojlab.herokuapp.com/userrecordsbyid (по дефолту запит виконується для користувача, для якого
User id: 1)

Щоб отримати список записів по певному користувачу в певній категорії, потрібно виконати POST request, щоб отримати по
якому користувачеві ви хочете шукати записи та в якій категорії ми будемо їх шукати, а тоді виконати запит:
https://backendprojlab.herokuapp.com/userrecordsincategory (по дефолту запит виконується для користувача, для якого
User id: 1, в категорії, для якої Category id: 1)

У файлі колекції Postman є запити для додавання користувача, запису, категорії, визначення користувача для перегляду
його записів та визначення категорії і користувача для перегляду його записів в категорії.

------------------------

Запуск локально:
1. Клонувати репозиторій з гіт

    $ git clone https://github.com/Aslanch1k/backendproj

2. Запускати будемо через контейнер Docker:

    1) Завантажуємо Docker з офіційного сайту: https://www.docker.com/
    2) Білдимо зображення командою:

    $ docker build --build-arg PORT=5000 . -t backandproj:latest

    3) Білдимо та запускаємо контейнер командами:

    $ docker-compose build
    $ docker-compose up

Також додані Валюти (для користувачів та витрат). При створенні користувача, якщо не вказати валюту, по дефолту
встановиться долар. При створенні запису, якщо не вказати валюту запису, буде використана та, що встановлена
в даних про користувача. Валюти додаються так само як і інші сутності, при використанні будь-якої валюти, яку ще не
додавали, буде помилка про неіснуючу валюту (окрім USD, вона створюється по дефолту).
