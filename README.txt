backendproj

Посилання на додаток: https://backendprojlab.herokuapp.com

Щоб отримати список категорій потрібно виконати запит: https://backendprojlab.herokuapp.com/categories

Щоб отримати список записів по певному користувачу, потрібно виконати запит: https://backendprojlab.herokuapp.com/user<user id>records (для прикладу створено запит для користувача для якого user id : 1. Запит за яким можна переглянути записи для користувача з user id:1 -  https://backendprojlab.herokuapp.com/user1records)

Щоб отримати список записів по певному користувачу в певній категорії, потрібно виконати запит: user<user id>recordsincategory<category id> (для прикладу створено запит для користувача для якого user id : 2, в категорії для якої category id:1. Запит за яким можна переглянути записи для користувача з user id: 2, в категорії для якої category id:1 - https://backendprojlab.herokuapp.com/user2recordsincategory1)

У файлі колекції Postman є запити для додавання користувача, запису та категорії.
